Raw - participants' raw data outputted frtom PsychoPy

Wrangled - participant data once it has been individually tidied by the data_wrangling script

Coded - wrangled participant data once it has also been accuracy coded. This is done manually with two columns added (No.Errors and accuracyScore)

Final - all coded participant data combined into datasets for each dependent measure with data exclusions applied, ready for final analyses
